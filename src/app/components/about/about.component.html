Welcome to Face Detect App

A startup project for Quick Face Detect from CCTV footage.
This application provides a quick and convenient way to Detect Suspect face from live streaming of cctv footage.

Technologies
  • Template pages with Angular 19 and TypeScript
  • RESTful API Backend using ASP.NET Core 9.0 Web API
  • Database using Entity Framework Core
  • Authentication/Authorization based on OIDC/OAuth2 and ASP.NET Core Identity
  • API Documentation using Swagger
  • Client-side built on Angular CLI
  • Theming using Bootstrap 5
Features
  • A simple, not "over-engineered" project for Quick Face Detection
